Provides the simulator component facade.

Used properties
de.renew.eagerSimulation boolean false Specifies whether the simulator should use eager simulation.
de.renew.simulatorClass class name (not set) Specifies the simulator class to use.
de.renew.simulatorMultiplicity int 1 Specifies the simulator multiplicity to use. This property is evaluated only if de.renew.simulatorClass is set to {@link de.renew.engine.simulator.ParallelSimulator}.
de.renew.simulatorMode int 1 Specifies simulator class and multiplicity to use. This property does not get evaluated if de.renew.simulatorClass is set.
Negative values
Use multiple sequential simulators. The absolute value determines the number of simulators.
0
Same as -1: Use one sequential simulator.
1 (default)
Use one concurrent simulator.
Positive values
Use multiple concurrent simulators. The absolute value determines the number of simulators.
